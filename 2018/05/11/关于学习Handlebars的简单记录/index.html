发表于 2018-05-11 | 分类于 技术 | | 热度: ℃
Handlebars 是 JavaScript 的一个语义模板库。项目前端框架Fish-desktop模版即是基于Handlebars。通常，一个页面js文件有与之相对应的hbs模板文件。这里简单记录下Handlebars的学习小结，是比较基础和初级的部分，主要是为了熟悉了一下它的语法和使用方式。

Handlebars简介

  • Handlebars 是 JavaScript 一个语义模板库，通过对view和data的分离来快速构建Web模板。它采用”Logic-less template”（无逻辑模版）的思路，在加载时被预编译，而不是到了客户端执行到代码时再去编译，这样可以保证模板加载和运行的速度。

  • 虽然是无逻辑，但还是有一些简单的逻辑比如if、each等。
