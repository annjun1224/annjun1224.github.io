javaWeb开发环境:一些简单前期准备配置

发表于 2018-03-14 | 分类于 技术 | | 热度: ℃
主要记录了javaWeb开发环境下一些必备工具的简单配置，包括jdk、maven、tomcat、mysql和idea的设置，算是基础的开发准备。

JDK

环境变量配置

  1. 系统变量→新建 JAVA_HOME 变量。
    变量值填写jdk的安装目录。
    如本机：C:\Program Files\Java\jdk1.8.0_131

  2. 系统变量→Path 变量→编辑
    在变量值最后输入 %JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

  3. 系统变量→新建 CLASSPATH 变量
    变量值填写 .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（注意最前面有一点）

  4. 检验是否配置成功 运行cmd 输入 java -version

Maven

环境变量配置

  1. 添加新的系统环境变量MAVEN_HOME，并设置其值为你安装的目录
    MAVEN_HOME= E:\tools\apache-maven-3.5.0
  2. 更新系统PATH变量，添加;%MAVEN_HOME%\bin;到尾部

测试配置是否成功

  • 打开命令行窗口，输入mvn -v，如果有maven 版本信息输出则证明配置成功，否则请查看自己配置路径等是否正确。
  • 注意：安装maven前请确保已安装JDK并成功配置其环境变量。

Tomcat

环境变量配置

  1. 变量名：CATALINA_HOME
  2. 变量值：tomcat安装路径，如：E:\tools\apache-tomcat-7.0.82

测试配置是否成功

IntelliJ IDEA

前期简单配置

进入画面之前-右下角configure

  • 添加jdk：
    • -project defaults-project structure
    • 在SDKs中+，找到jdk目录，添加，应用。
    • 在project中选择对应jdk。

  • 添加maven
    -settings-搜索maven，添加maven解压目录。

  • 快捷键设置
    -settings-keymap，可以选择eclipse。

  • 字体大小

  • 创建
    • create-maven-勾选create from archetype。
    • 选择maven-archetype-webapp。
    • 下载好后，main下创建java文件夹，右键-mark directory as-sources root。
  • 添加tomcat
    • 点击右上方菜单栏下箭头-edit-tomcat server-local-configuer，选择tomcat目录。
    • 重命名-选择deployment上面的+ 选择artifact-war。
    • 点绿箭头可启动tomcat。
  • 配置mybatis插件mybatis-plugin
    • file-settings-plugin-browse repositories。
    • 查找mybatis plugin install。
    • 其他插件也可如此配置，如lombok插件。
